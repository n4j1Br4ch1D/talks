#  Beyond the Hype: Rethinking SPAs in a Different Way

*presented At:* 
- [DEVOXX.MA](https://devoxx.ma/talk/beyond-the-hype-rethinking-spas-in-a-different-way)

SPAs have transformed web development by offering seamless user experiences but have also introduced significant complexity.

In this talk, we will explore the ongoing excitement and hype surrounding emerging alternatives such as HTMX, Inertia, Livewire(Laravel), Turbo(Rails), Liveview(Phoenix Elixir), and Asp.net(Blazor), and how they promise to reduce complexity and enhance productivity compared to mainstream approaches led by React, Vue, Angular, and Svelte.

We will then examine how the Java Spring Boot ecosystem is embracing these innovations(Vaadin/HTMX/Inertia).
